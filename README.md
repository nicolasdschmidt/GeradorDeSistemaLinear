# Gerador de Sistema Linear

## Uso
	node matriz.js <tamanho>

Exemplo:
	
	node matriz.js 5

Gera algo como:

	5
	0 -16 -15 -10 15 21 
	-5 0 5 2 -15 -86 
	3 14 0 -20 -3 -23 
	-16 12 20 0 15 28 
	18 1 0 -19 0 -37 

Cuidado com a linha vazia no fim do arquivo, porque dependendo do seu leitor, pode dar erro.
